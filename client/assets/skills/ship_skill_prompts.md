# Промты для иконок активных способностей кораблей

Целевой размер: **512×512 px**, PNG с **прозрачным фоном**.

Стиль: sci-fi UI badge icon, painterly stylized game UI icon, AAA mobile MMO ability icon,
centered composition, circular border ring, isolated on transparent background,
no characters, no text, no letters, easy-to-read silhouette at small size.
