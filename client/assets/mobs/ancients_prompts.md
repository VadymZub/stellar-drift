# ДРЕВНИЕ — кристаллическая раса. Промты для 13 моделей + Апофис + Кристальный щит

## Лор и роль

**Фракция:** Древние (The Ancients)  
**Внутренний код:** `ancient_01` … `ancient_12` + `apophis` (boss)  
**Уровни:** 18–30 (финальная угроза, элита)  
**Зоны:** глубокий космос, аномальные секторы, зоны вне корпоративного контроля

Древние — доиндустриальная космическая цивилизация, существовавшая за миллионы лет до людей.
Их «корабли» — живые кристаллические конструкции, питаемые энергией звёзд.
Они не строят — они выращивают. Их оружие — поглощение энергии, гравитационные мины,
щитовое высасывание и временны́е аномалии.

Апофис — их проснувшийся страж, тысячелетиями дрейфовавший в глубоком космосе.

**Тактика:** поглощение щитов → минное заграждение → кристаллический рой → финальный удар.  
**Особенности:** антищитовые мины, дрейнеры щита, кристаллические дроны, гравитационные ловушки.

## Анимация Апофиса в Phaser 4

### Концепция: слоистая анимация из нескольких объектов

Вместо одного спрайта — несколько слоёв:

```
Апофис = 4 Phaser-объекта:
  [0] ancient_12.png        — статичное ядро (глубина 40)
  [1] ring_outer.png        — внешнее кольцо, вращение +0.3 рад/с
  [2] ring_mid.png          — среднее кольцо, вращение -0.5 рад/с
  [3] ring_inner.png        — внутреннее кольцо, вращение +0.8 рад/с
```

**Идея кольца:** отдельный PNG каждого кольца — дуга кристаллов без фона.
В Phaser rotate через `sprite.rotation += delta * speed`.

### Реализация пульса ядра (tween):
```javascript
this.scene.tweens.add({
  targets: core,
  alpha: { from: 0.85, to: 1.0 },
  scaleX: { from: 1.0, to: 1.04 },
  scaleY: { from: 1.0, to: 1.04 },
  duration: 1200,
  yoyo: true,
  repeat: -1,
  ease: 'Sine.easeInOut'
});
```

### Фаза 2 (hp < 40%):
- Заменить кольца на `ancient_13.png` (разбитый спрайт)
- Осколки: 8 маленьких `shard.png` вращающихся по эллиптическим орбитам
- Tween ускорение: `rotSpeed *= 2.5`
- Tint ядра: `core.setTint(0xff6600)` (оранжевый накал)
- Particle emitter: искры от трещин
