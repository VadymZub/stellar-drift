# Rank System — Stellar Drift

Ранговая система основана на **рейтинге** игрока внутри корпорации:
`rating = XP × 0.4 + Honor × 0.6`

Ранги 1–3 — фиксированные слоты (буквально N мест на всю корпорацию).
Ранги 4–20 — процент от активных игроков (top X%).

## Реализация в коде

**Определение тира по rank.id** — `src/entities/Player.js`:
```js
export function rankTier(id) {
  if (id <= 3)  return 1;  // Supreme Command
  if (id <= 5)  return 2;  // Admiralty
  if (id <= 9)  return 3;  // Senior Officers
  if (id <= 13) return 4;  // Junior Officers
  if (id <= 15) return 5;  // Senior NCO
  if (id <= 17) return 6;  // Junior NCO
  return 7;                // Enlisted
}
```

**Отображение** — nameplate над кораблём: иконка 22×22 (NORMAL blend) + имя игрока, обновляется каждый кадр в `Player.update()`.

**Расчёт ранга** — `src/ranking.js`: `calculateRating(xp, honor)` → `getRank(rating, allRatings)`.

**Цвета** — `RANK_TINTS` в `src/entities/Player.js`, экспортирован для внешнего использования.
