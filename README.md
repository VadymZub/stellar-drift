# Stellar Drift — игровой прототип (Phase 1)

Браузерная космо-MMO в духе Andromeda5/DarkOrbit. Это первый играбельный
вертикальный срез: одна home-карта, корабль Wisp, бой с мобами.

## Запуск

ES-модули не работают по `file://` (CORS), нужен HTTP-сервер:

```powershell
cd C:\Work\stellar-drift\client
./run.ps1
```

Скрипт поднимет `python -m http.server 8080` и откроет http://localhost:8080.
Сборка не нужна — Phaser 4.1.0 подгружается с CDN как ES-модуль.

## Что уже работает

- **Движение Andromeda5-style:** ЛКМ по пустому космосу → курс к точке;
  клик по янтарной стрелке-шеврону впереди корабля → форсаж ×2 со сжиганием
  щита (в безопасной зоне — бесплатно).
- **Бой:** клик по мобу или `TAB` — захват цели; авто-огонь плазмой в радиусе;
  снаряды, урон с пробитием щита, всплывающие числа урона.
  **Реализованы механики уклонения (Evasion) и пробития (Penetration).**
- **VFX Взрывов:** реализованы все 6 классов взрывов (micro/small/medium/large/huge/mega) с автоматической привязкой к размеру цели. 28-кадровая анимация (рост → распад → кольцо).
- **Щит/корпус** с регенерацией щита после паузы в бою.
- **Мобы:** 22 уникальных шаблона (Корсары, Синдикат, Древние + супер-босс Аннигилятор), скейл по уровню, сложный AI: агро → подлёт → огонь, респаун через 8 сек.
- **Мини-боссы и Боссы:** спец-механика — **фазы ярости** (ниже 40 % корпуса — ускоренный огонь/движение, красная подсветка), **телеграф-AoE** (нужно уйти из круга до детонации), **дроп ⭐ Звёздного золота**.
- **Джампгейты:** функциональные порталы по краям карты для прыжков между секторами с проверкой уровня доступа.
- **HUD:** статы игрока, панель цели (с индикатором «ЯРОСТЬ»), event-лог, кредиты, ⭐ Звёздное золото, **уровень пилота + XP-бар**.
- **Прогрессия пилота:** опыт капает с убийств мобов (только PvE), уровень
  растёт по кривой `40·L² + 13·max(0,L−25)³`.
- **Прогрессия корабля:** уровни 1-10 для каждого корпуса с бонусами к HP/урону/скорости и расчетом стоимости прокачки в кредитах.
- **Гараж — витрина кораблей (таб «КОРАБЛИ»):** весь модельный ряд 3×3.
  Купленные активны/выбираемы, статы и требования (level-gate, цена). Базовые покупаются за
  **кредиты**, prestige — за **⭐** + гибрид-гейт (победа корпы в сезоне + ур.45 + реп 80%).
- **Оборудование и Склад:** реализована система слотов модулей (оружие, щиты, двигатели). Статы корабля суммируются на основе надетых модулей.
- **i18n** с первой строки: весь текст через `i18n.t()`, локаль `locales/ru.json`.

> **DEV-хоткеи для проверки витрины** (в релизе убрать): `0` — +1 уровень
> пилота (на 45+ открывает гейт престижа); `9` — +100 000 кр и +500 ⭐.

## Управление

| Действие | Ввод |
|---|---|
| Лететь в точку | ЛКМ по космосу |
| Форсаж вкл/выкл | клик по стрелке-шеврону |
| Захватить цель | клик по кораблю врага |
| Следующая цель | `TAB` |
| Снять цель | `ESC` |
| Инвентарь | `I` |
| Гараж | `G` |
| Карта галактики | `M` |

## Структура

```
client/
  index.html            точка входа (<script type=module>)
  run.ps1               локальный статик-сервер       
  locales/ru.json       весь UI-текст (MVP — только ru)
  src/
    main.js             конфиг Phaser, список сцен    
    constants.js        палитра + баланс (статы из content-scope)
    i18n.js             i18n.t()
    scenes/             Boot / Game / Hud / Garage / Inventory / Map
    entities/           Player / Mob / Projectile / Loot    
    systems/            Movement (click-to-move + форсаж) / Minimap
  assets/               wisp, мобы, стрелки, карты
```
